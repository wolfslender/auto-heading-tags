# Auto Heading Tags Plugin for WordPress

## Description

The Auto Heading Tags Plugin is a simple yet powerful tool designed to automate the process of adding heading tags (H1, H2, H3, etc.) in WordPress posts. Originally developed for personal use on the news website Infocero, this plugin is now available for anyone looking to streamline their content creation process. It helps improve SEO and readability by automatically structuring your post content with the appropriate HTML heading tags.

## Features

- Automatically generates H1, H2, H3, and other heading tags based on your post's content.
- Seamless integration with WordPress, with no need for complex configurations.
- Optimizes SEO by ensuring proper heading structure within posts.
- Lightweight and fast, ensuring no impact on your site's performance.
- Developed initially for Infocero, a news website, but works for any type of WordPress blog or site.

## Installation

1. Download the plugin files.
2. Upload the plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the **Plugins** menu in WordPress.
4. Enjoy automated heading tags on your posts!

## Usage

Once activated, the plugin automatically scans your post content and applies the appropriate heading tags. There’s no need for any manual intervention—just focus on writing your content, and let the plugin take care of the rest.
